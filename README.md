# 加班数据提取工具

这是一个图形界面应用程序，用于从Excel文件中提取指定研究室的加班数据，并生成格式化的结果文件。

## 目录结构

```
.
├── src/                 # 源代码目录
│   ├── main.py          # 应用程序入口
│   ├── gui.py           # 图形界面模块
│   └── core.py          # 核心业务逻辑模块
├── start_gui.bat        # Windows启动脚本
├── requirements.txt     # 依赖包列表
├── .gitignore           # Git忽略文件配置
└── README.md            # 项目说明文档
```

## 功能特点

- 图形界面操作，简单易用
- 支持选择Excel源文件（.xlsx或.xls）
- 支持拖放文件操作
- 可选择结果文件保存位置
- 可选择要提取的一个或多个研究室
- 自动生成格式化的Excel结果文件

## 安装依赖

在运行应用程序之前，请确保安装了所需的依赖包：

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行GUI应用程序：

   方法一（命令行）：

   ```bash
   python src/main.py
   ```

   方法二（Windows用户）：
   双击运行 `start_gui.bat` 批处理文件
2. 在GUI界面中：

   - 点击"选择源文件"按钮或直接将Excel文件拖放到指定区域来选择源文件
   - 点击"选择保存位置"按钮选择结果文件的保存目录（可选，默认为源文件所在目录）
   - 勾选要提取数据的研究室复选框（可多选）
   - 点击"开始处理"按钮执行数据提取
   - 查看处理日志了解处理进度
   - 处理完成后，结果文件将保存在指定位置，包含所有选定研究室的数据

## 原始命令行版本

如果需要使用原始的命令行版本，可以运行：

```bash
python src/main.py
```

## 系统要求

- Python 3.6或更高版本
- Windows、macOS或Linux操作系统

## 依赖说明

- `pandas`: 用于数据处理
- `openpyxl`: 用于Excel文件读写
- `PyQt5`: 用于图形界面

## 注意事项

- 源Excel文件必须包含"研究室"、"姓名"、"周末加班记录"、"周末加班奖励总额"等列
- 确保源文件格式正确，避免数据处理错误
